# Просмотр меню
------Небольшое изменение

## Акторы:


- Пользователь
- Приложение
- Бекенд


## Цель:


Посмотреть меню пиццерии.


## Основной сценарий:


1. Пользователь заходит на сайт.
2. Приложение запрашивает у бекенда список товаров.
3. Бекенд возвращает данные товарах, имеющихся в системе.
4. Приложение открывает Экран меню.
5. Пользователь выбирает нужный товар.
6. Приложение запрашивает у бекенда данные о выбранном товаре.
7. Бекенд возвращает данные о выбранном товаре.
8. Приложение открывает Экран товара.


## Альтернативный сценарий:


3а. Если бекенд не может вернуть данные о товарах, приложение отображает сообщение об ошибке и предлагает повторить попытку позже.


6а. Если пользователь не выбрал ни один товар, приложение отображает сообщение об ошибке и предлагает повторить попытку.


7а. Если бекенд не может вернуть данные о выбранном товаре, приложение отображает сообщение об ошибке и предлагает повторить попытку позже.
